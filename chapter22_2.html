Chapter 22: Smoothing Using Orthogonal Functions

Mathematical Foundations of Function Approximation

🎯 Why Orthogonal Functions?

Orthogonal function smoothing provides an alternative to kernel methods for nonparametric curve estimation. Instead of local averaging, we project the unknown function onto a finite-dimensional space spanned by orthogonal basis functions.

🌟 Key Advantages

Global approach: Uses all data points for each coefficient

Computational efficiency: Fast transforms (FFT, wavelets)

Theoretical foundation: Clear bias-variance decomposition

Adaptivity: Can handle functions with varying smoothness

🔍 Basic Idea

f(x) ≈ ∑j=1J θjφj(x)

Approximate function f using finite linear combination of orthogonal basis functions {φj}

📊 Orthogonal Functions and Projections

🎯 Hilbert Space Setup

Let ℋ be a Hilbert space with inner product ⟨f, g⟩

Orthonormal basis: ⟨φⱼ, φₖ⟩ = δⱼₖ
Function expansion: f = ∑ⱼ θⱼ φⱼ
Fourier coefficients: θⱼ = ⟨f, φⱼ⟩

📐 Projection Theorem

The finite approximation fJ(x) = ∑j=1J θⱼφⱼ(x) minimizes:

‖f - g‖² over all g ∈ span{φ₁, ..., φⱼ}

Optimality: Best approximation in the given subspace

🎼 Examples of Bases

  • Fourier: {1, cos(jx), sin(jx)}
  • Wavelets: {ψⱼ,ₖ(x) = 2^(j/2)ψ(2ʲx-k)}
  • Polynomials: {1, x, x², ...}
  • Splines: B-spline basis functions

🔧 Inner Products

  • L²[0,2π]: ⟨f,g⟩ = ∫₀²π f(x)g(x)dx
  • Weighted: ⟨f,g⟩ = ∫ f(x)g(x)w(x)dx
  • Discrete: ⟨f,g⟩ = ∑ᵢ f(xᵢ)g(xᵢ)

🎯 Estimation Methodology

📊 Method of Moments Estimator

Given observations (X₁, Y₁), ..., (Xₙ, Yₙ) where Yᵢ = f(Xᵢ) + εᵢ:

θ̂ⱼ = (1/n) ∑ᵢ₌₁ⁿ Yᵢ φⱼ(Xᵢ)

f̂ⱼ(x) = ∑ⱼ₌₁ᴶ θ̂ⱼ φⱼ(x)

Interpretation: Empirical version of Fourier coefficients

📈 Bias and Variance Decomposition

For fixed design points and Gaussian noise εᵢ ~ N(0, σ²):

  • Bias: E[f̂ⱼ(x)] - f(x) = f(x) - fⱼ(x)
  • Variance: Var(f̂ⱼ(x)) = (σ²/n) ∑ⱼ₌₁ᴶ φⱼ²(x)
  • MSE: |f(x) - fⱼ(x)|² + (σ²/n) ∑ⱼ₌₁ᴶ φⱼ²(x)

⚖️ Bias-Variance Tradeoff

Small J: Low variance, high bias (underfit)

Large J: High variance, low bias (overfit)

Optimal J: Minimizes MSE = Bias² + Variance

🎼 Fourier Series Methods

🌊 Trigonometric Basis

For functions on [0, 2π], the trigonometric basis is:

φ₀(x) = 1/√(2π)
φ₂ⱼ₋₁(x) = cos(jx)/√π
φ₂ⱼ(x) = sin(jx)/√π

🔄 Fourier Series Approximation

fⱼ(x) = a₀/2 + ∑ⱼ₌₁ᴶ/² [aⱼcos(jx) + bⱼsin(jx)]

Coefficients:

  • a₀ = (2/π) ∫₀²π f(x)dx
  • aⱼ = (2/π) ∫₀²π f(x)cos(jx)dx
  • bⱼ = (2/π) ∫₀²π f(x)sin(jx)dx

⚡ Fast Fourier Transform

When design points are equally spaced, use FFT for O(n log n) computation

Discrete Fourier Transform:
X_k = ∑ⱼ₌₀ⁿ⁻¹ xⱼ e^(-2πijk/n)

🎯 Applications

  • Periodic signals: Natural choice
  • Time series: Spectral analysis
  • Image processing: 2D Fourier methods
  • Signal denoising: Low-pass filtering

🌊 Wavelet Methods

🔍 Wavelet Basis

Wavelets provide localization in both time and frequency:

Scaling functions: φⱼ,ₖ(x) = 2^(j/2) φ(2ʲx - k)
Wavelets: ψⱼ,ₖ(x) = 2^(j/2) ψ(2ʲx - k)

Where j controls scale and k controls location

🏗️ Multiresolution Analysis

Decompose function into multiple scales:

f(x) = ∑ₖ αⱼ,ₖ φⱼ,ₖ(x) +
∑ⱼ'=1ᴶ ∑ₖ βⱼ',ₖ ψⱼ',ₖ(x)
  • αⱼ,ₖ: Approximation coefficients
  • βⱼ,ₖ: Detail coefficients

✂️ Thresholding

Key advantage: can threshold detail coefficients

Hard: threshold(x,λ) = x·I(|x| > λ)
Soft: threshold(x,λ) = sign(x)(|x|-λ)₊

Effect: Adaptive smoothing, preserves edges

🎯 Wavelet Advantages

  • Adaptivity: Automatically adapts to local smoothness
  • Edge preservation: Good for functions with jumps
  • Computational efficiency: Fast wavelet transform
  • Sparsity: Many coefficients are small

🌿 Splines as Projections

📐 Spline Spaces

A spline of degree d with knots t₁ < ... < tₖ is a piecewise polynomial that is (d-1) times continuously differentiable

🔧 B-spline Basis

Spline space can be represented using B-spline basis {B₁, ..., Bₖ₊ₐ₊₁}

s(x) = ∑ⱼ₌₁ᴷ⁺ᵈ⁺¹ cⱼ Bⱼ(x)

Properties: Local support, numerical stability

🎯 Smoothing Splines

Minimize penalized sum of squares:

∑ᵢ₌₁ⁿ (Yᵢ - f(Xᵢ))² + λ ∫ (f''(x))² dx

Solution: Natural cubic spline with knots at data points

Parameter λ: Controls smoothness vs fidelity tradeoff

📊 Model Selection

✅ Cross-Validation

Choose J to minimize cross-validation error:

CV(J) = (1/n) ∑ᵢ₌₁ⁿ (Yᵢ - f̂ⱼ⁽⁻ⁱ⁾(Xᵢ))²

Where f̂ⱼ⁽⁻ⁱ⁾ is estimate without i-th observation

⚡ Generalized CV

For orthogonal bases, efficient GCV formula:

GCV(J) = n∑ᵢ(Yᵢ - f̂ⱼ(Xᵢ))² / (n - J)²

Advantage: No need to refit n times

📈 Information Criteria

Criterion Formula Penalty
AIC n log(RSS/n) + 2J Linear in J
BIC n log(RSS/n) + J log(n) Stronger penalty
Mallows Cp RSS/σ² + 2J - n Unbiased risk estimate

📏 Risk Properties and Rates

🎯 Minimax Rates

For functions in Sobolev space W^s (s bounded derivatives):

inf sup E‖f̂ - f‖² ≍ n^(-2s/(2s+1)) f̂ f∈W^s

Achievability: Both wavelets and smoothing splines achieve this rate

🔄 Adaptive Methods

Achieve near-optimal rates without knowing smoothness s:

  • Wavelet thresholding: Donoho-Johnstone estimators
  • Model selection: Cross-validation, penalized methods
  • Stein's unbiased risk: SURE-based selection

📊 Rate Comparison

Method Rate Adaptive?
Fourier (known s) n^(-2s/(2s+1)) No
Wavelet threshold n^(-2s/(2s+1)) log n Yes
Smoothing splines n^(-2s/(2s+1)) With GCV

⚠️ Practical Considerations

🚫 Boundary Effects

Problem: Orthogonal functions may not handle boundaries well

Solutions:

  • Boundary correction methods
  • Reflection/extension techniques
  • Boundary-adapted bases

⚡ Computational Issues

Advantages:

  • FFT makes Fourier methods O(n log n)
  • Fast wavelet transforms
  • Splines lead to banded systems

Disadvantages:

  • May need many terms for good approximation
  • Choice of basis crucial

🎯 Method Selection Guidelines

  • Fourier: Periodic functions, smooth signals
  • Wavelets: Functions with edges/jumps, varying smoothness
  • Splines: General purpose, good boundary behavior
  • Polynomials: Simple smooth functions, low order

🌟 Summary and Key Takeaways

🎯 Main Messages

  • Global approach: Orthogonal functions use all data for each coefficient
  • Basis selection matters: Choose basis appropriate for function class
  • Computational efficiency: Fast transforms enable large-scale applications
  • Adaptivity possible: Thresholding and model selection provide adaptation
  • Optimal rates: Can achieve minimax rates for smooth function classes
  • Complementary to kernels: Different strengths for different problems

🚀 The Big Picture

Orthogonal function methods provide a powerful alternative to kernel-based smoothing, especially when computational efficiency, global structure, or adaptation to varying smoothness is important.

Key advantages: Fast computation, theoretical guarantees, natural handling of periodic/structured functions

Key challenges: Boundary effects, basis selection, global nature can miss local features
