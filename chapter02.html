P(A|B) = P(A ‚à© B) / P(B), provided P(B) > 0

üéØ Definition

Independence: Knowledge of one event doesn't change probability of another

A and B are independent if: P(A ‚à© B) = P(A) √ó P(B)

‚ö†Ô∏è Common Misconceptions

‚ùå Gambler's Fallacy

Myth: "If coin shows 5 heads, tails is due"

Reality: Each flip is independent

Truth: P(T on flip 6) = 0.5 regardless of history

‚ùå Confusion of the Inverse

Myth: P(A|B) = P(B|A)

Example: P(test+|disease) ‚â† P(disease|test+)

Need: Bayes' theorem to relate them

‚ùå Base Rate Neglect

Problem: Ignoring prior probabilities

Example: Medical test with low disease prevalence

Solution: Always consider base rates in Bayes' theorem

‚ùå Independence Confusion

Myth: Disjoint events are independent

Reality: Usually opposite is true

Example: Can't rain and be sunny simultaneously

üõ†Ô∏è Problem-Solving Strategy

üìã Step-by-Step Approach

  1. Define sample space: What are all possible outcomes?
  2. Identify events: What exactly are we calculating?
  3. Check for independence: Does one event affect another?
  4. Choose appropriate formula: Basic probability, conditional, Bayes?
  5. Calculate step by step: Show your work
  6. Verify answer: Does it make intuitive sense?

üß† Key Questions to Ask

  • "What information is given?" - Suggests conditional probability
  • "Are events mutually exclusive?" - Use addition rule
  • "Do we need to update beliefs?" - Use Bayes' theorem
  • "Is this a counting problem?" - Use combinatorics
