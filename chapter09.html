üîÑ What is the Bootstrap?

üéØ The Revolutionary Idea

Bootstrap principle: Use the data to simulate the data generating process

üîç The Problem

üí° The Insight

Strategy: Treat sample as population, resample from it

üé≤ Simple Analogy

Bootstrap answer: Put balls back, resample with replacement!

‚ö†Ô∏è Bootstrap Limitations and Failures

‚ùå Dependent Data

Problem: Time series, spatial data violate independence

Solution: Block bootstrap, model-based bootstrap

Example: Stock prices have temporal dependence

‚ùå Extreme Value Statistics

Problem: Bootstrap can't extrapolate beyond sample

Issue: Sample max ‚âà population max

Alternative: Extreme value theory, parametric methods

‚ùå Small Sample Sizes

Problem: Bootstrap needs "large enough" n

Rule of thumb: n ‚â• 20-30

Issue: Limited diversity in bootstrap samples

‚ùå Heavy-Tailed Distributions

Problem: Infinite variance breaks theory

Example: Cauchy distribution

Symptom: Bootstrap distribution doesn't stabilize

üö´ When NOT to Use Bootstrap

  • Time series without modification: Use block bootstrap instead
  • Estimating tail probabilities: Bootstrap underestimates
  • Very small samples: n < 20 unreliable
  • Boundary problems: Parameters near bounds
  • Model selection: Can give overly optimistic results
